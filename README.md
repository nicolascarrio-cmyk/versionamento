# # Atividade: Manipulação de Vetores em JavaScript

Este repositório contém uma série de exercícios práticos desenvolvidos em JavaScript para consolidar conceitos de estruturas de repetição, condicionais e manipulação de arrays (vetores).

## 🚫 Restrição de Desenvolvimento
Para exercitar a lógica de programação puramente algorítmica, todos os programas foram construídos **sem a utilização de funções nativas de arrays do JavaScript** (tais como `.push()`, `.filter()`, `.join()`, `.reduce()` ou o objeto `Set`). Toda a inserção, busca, filtragem e formatação de dados para exibição foram feitas utilizando laços de repetição manuais (`for`).

## 📋 Exercícios Desenvolvidos

* **Exercício 1 (Soma de Pares):** Lê 10 números inteiros informados pelo usuário via `prompt`, armazena-os em um vetor e, ao final, calcula e exibe através de um `alert` apenas a soma dos números que são pares.
* **Exercício 2 (Números Aleatórios):** Preenche de forma automatizada um vetor de 7 posições com números inteiros gerados aleatoriamente pelo computador (intervalo de 1 a 100) e exibe o resultado final na tela usando `alert`.
* **Exercício 3 (Contador de Ocorrências):** Solicita 10 números inteiros ao usuário e os armazena. Em seguida, pede um número adicional para busca e conta de forma manual quantas vezes esse número aparece no vetor original, mostrando o resultado em um `alert`.
* **Exercício 4 (Remoção de Duplicados):** Recebe 10 números do usuário. Através de uma estrutura de laços aninhados, identifica os números repetidos e cria um novo vetor contendo apenas os valores únicos, exibindo a comparação entre o vetor original e o vetor limpo em uma caixa de `alert`.

## 🛠️ Tecnologias Utilizadas
* JavaScript (ES6+)
* Git & GitHub (para controle de versão e deploy do código)

## 🚀 Como Executar os Arquivos
Como todos os scripts utilizam exclusivamente as funções `prompt()` e `alert()` para interação direta com a interface do usuário, eles devem ser executados no ambiente do navegador:
1. Abra qualquer página da web no seu navegador.
2. Abra as ferramentas de desenvolvedor (`F12` ou clique com o botão direito e selecione **Inspecionar**).
3. Vá até a aba **Console**.
4. Copie o código do exercício desejado, cole ali dentro e pressione `Enter`. 
5. As caixas de diálogo interativas (`prompt` para digitar e `alert` com os resultados) aparecerão diretamente na sua tela.
